# Machine-Learning-CW2

This repository contains the files for my Machine Learning Coursework 2.

## Repository Contents

- `MLCoursework2.ipynb`  
  The main Jupyter Notebook containing the code for data preprocessing, model training, hyperparameter tuning, evaluation, and result generation.

- `project_results.zip`  
  A compressed file containing the saved outputs/results generated from the notebook.

## Notes

The notebook includes the full workflow used for the coursework, including model comparison and tuned model evaluation. The ZIP file is included so that the generated results can be reviewed without needing to rerun the full notebook.

## How to Use

1. Open `MLCoursework2.ipynb` in Google Colab, or VS Code.
2. Run the notebook cells in order if you want to reproduce the workflow.
3. Open or extract `project_results.zip` to view the saved result files.
